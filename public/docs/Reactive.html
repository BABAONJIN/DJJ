(static) prop(type, name, val)

prop() can be used to define a new property.

The following types are allowed:

  1. "persist" is a property that will be stored in the browser's localStorage. It can be changed by calling update().
  2. "proxy" is a property that belongs to a child object.
  3. "ro" is a property that cannot be changed.
  4. "rw" is a property that can be changed by the "update()" method.

The code below cannot be used to update any prop() property. A change must go through update().

this.some_property = 'new value';
Parameters:
Name Type Description
type String

either "persist", "proxy", "ro" or "rw".
name String

The name of the property
val Any

Either a function or default value.

(static) subscribe(cb)

subscribe() is a special case of on() to listen for "update" events. This method exists because it is used by Svelte stores.

The function passed to this method will also be called right away, with the current object as parameter.

(static) update(params)

update() can be used to change the properties defined by prop(). This method will asynchronously emit the "update" event once all properties have been updated.

Any key in "params" that does not map to a defined prop() will be ignored.

Parameters:
Name Type Description
params Object

A map between property name and value.
