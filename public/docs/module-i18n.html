(static) l(lexicon, …vars)

l() is used to translate strings. The strings can contain "%1", "%2", ... which will be replaced by the additional variables.

Parameters:
Name Type Attributes Description
lexicon String

A string that you want to translate.
vars String <repeatable>

A list of variables to put into the translated string.
